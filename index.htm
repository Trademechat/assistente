Política de Privacidade

Última atualização: 30/07/2025

1. Coleta de Informações

Este assistente não coleta dados pessoais sensíveis, como nome, endereço ou documentos. As interações são processadas apenas para fornecer respostas e podem ser registradas de forma anônima para fins de melhoria do serviço.

2. Uso das Informações

As informações fornecidas são utilizadas exclusivamente para:

  • Processar e responder às solicitações dos usuários.
  • Melhorar a experiência do usuário e a qualidade das respostas.
  • Garantir a segurança e o correto funcionamento do assistente.

3. Compartilhamento de Dados

Não compartilhamos dados de usuários com terceiros, exceto quando exigido por lei.

4. Segurança

Adotamos medidas técnicas e administrativas adequadas para proteger os dados contra acesso não autorizado, alteração ou destruição.

5. Direitos do Usuário

O usuário pode, a qualquer momento, solicitar a exclusão de seus dados de interação, quando aplicável, entrando em contato pelo canal oficial.

6. Alterações nesta Política

Podemos atualizar esta política periodicamente. As alterações serão comunicadas com uma nova data de atualização.

7. Contato

Para dúvidas sobre esta Política de Privacidade, entre em contato pelo e-mail: seuemail@dominio.com
